User Manual

Notes Module

Create a note: In the tree view, tap “get items” > notes, you will have a list of notes associated with selected node. Tap + to add a new note. A note must have a title.

Open a note:In the tree view, tap “get items” > notes, you will have a list of notes associated with selected node. Select your note.

Update a note:

To update a note, open it, edit its content or title, then hit “save”. Decryption/Encryption automatically save your note.

Delete a note:

To delete a note, open it, then select the menu item “delete” in the note viewer, then confirm the deletion by hitting “yes” in the confirm dialog.

Encrypt a note:

To encrypt a note, use the “encrypt/decrypt” button

Decrypt a note:

To decrypt a note, use the “encrypt/decrypt” button

Note: When you open an encrypted note, the text you see in the note viewer is independent of the encrypted content. You can, for example, edit the text of an encrypted note to anything you want then save it without altering the encrypted content.

Todo list

Create/select a todo list:

select a node

select the “get items” button

select “Todo list”

Add items to a todo list:

select a todo list

use the “+” button

if you have selected a todo item, this will create a new item right after the selected item

Add subtask to todo list:

make sure the subtask item is right below the parent task item

use the “→” button

Remove item from todo list:

select a todo item

use the “-” button

this will also delete subtasks

“←” button :

use this button to unparent a subtask

the item will be placed right below the parent item

you can use ↑ and ↓ to rasie or lower the position of the selected item

if you have selected an item, you can use the keyboard button to dismiss the keyboard. Note that you must select any item before dismissing the keyboard.

Nodes :

Add node:

from the tree view, tap the “+tree” button, specify a name for the node,  then tap enter, the node will be a child of the current node, or a root if the view is empty.

Delete node:

from the tree view, open the menu, tap “delete node”, then tap “yes” to confirm.

Deleting a node will delete all associated modules data (notes, reminders, todos, etc).

goto node:

from the text view, enter a number in the “goto node” field then tap the done (V) button.

rebind node parent:

go to child node: tap the child node of the node to go to the child of the node

go to parent node:  tap the node to go to the parent node
